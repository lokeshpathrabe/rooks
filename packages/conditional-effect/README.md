### Installation

```
npm install --save @rooks/use-conditional-effect
```

### Importing the hook

```javascript
import useConditionalEffect from "@rooks/use-conditional-effect"
```

### Usage

```jsx
function Demo() {
  useConditionalEffect();
  return null
}

render(<Demo/>)
```
